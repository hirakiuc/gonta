# gonta

gonta is a simple slack bot in golang.

## HowToBuild

This project use [Masterminds/glide](https://github.com/Masterminds/glide) for dependency management.

```
$ make vendor_get
$ make
```

## HowToRun

```
$ export SLACKA_API_TOKEN='your slack api token'
$ make run
```
